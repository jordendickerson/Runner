# ENDLESS RUNNER
## An endless runner game made in Pygame

### **THE GAME**
As you run, you must stay on the green tiles. If you touch the red or orange, game over!
Shoot orange tiles to break them!

### **CONTROLS**
*Left and Right arrow keys* - Move left and right
*Up Arrow Key* - Jump
*Down Arrow Key* - Dive
*Space* - Shoot
